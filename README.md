## 📌 Descrição

Este projeto tem como objetivo desenvolver um sistema de gestão agrícola utilizando Python, aplicando conceitos de estruturas de dados, manipulação de arquivos e lógica de decisão.

O sistema permite:

- Coleta de dados do solo
- Análise de nutrientes (NPK)
- Avaliação de pH e umidade
- Recomendação de irrigação
- Suporte a múltiplas culturas

##  Tecnologias utilizadas

- Python
- JSON

##  Como executar

```bash
python scripts/sistema_agro.py
